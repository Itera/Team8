**Hash:** 1d2e3f4  
**Date:** 2026-04-16  
**Author:** marcin  

## Summary

Introduced the `/development_history` page and its detail subpages. Every
merge to `main` will automatically generate a markdown file and update the
change manifest via a GitHub Actions workflow, keeping the history page
current without manual effort.

## Changes

- Added `react-router-dom` to `frontend/`
- Created `frontend/src/views/DevelopmentHistory.tsx` listing all merges
- Created `frontend/src/views/ChangeDetail.tsx` rendering individual markdown
  files
- Created `frontend/src/matrix.css` with Matrix-themed styles
- Created `frontend/public/changes/index.json` as the manifest
- Seeded ten historical markdown entries under `frontend/public/changes/`
- Created `.github/workflows/generate-change-entry.yml` for automatic updates
- Updated routing in `App.tsx` and `main.tsx`

## Notes

The Matrix theme uses `Courier New` / monospace fonts, `#00ff41` green on
`#000000` black, and a subtle scanline animation to evoke late-1980s terminal
aesthetics. No external style libraries were added.
